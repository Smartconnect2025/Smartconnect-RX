The MFA system in Greenwich RX doesn't use an external API key. It's all built with open-source packages that work locally:

otplib — generates TOTP secrets and verifies codes. No API key needed, it's a pure crypto library.
qrcode — generates QR code images locally. No API key needed.
SESSION_SECRET — this is the only secret needed, and it's just an environment variable you set yourself (any random string). It's used to encrypt the MFA secrets stored in the database.
So for the AIM RX project, the only thing needed is:
