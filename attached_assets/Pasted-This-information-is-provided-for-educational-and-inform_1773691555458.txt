Expectation: The frontend sends a POST request with a verification code; the server processes it and returns a JSON object like { "success": true }.
Reality: The server is returning an HTML document containing an error message. When the frontend code tries to run JSON.parse() on this HTML, it crashes because HTML is not valid JSON. This is why the console logs [MFA] Non-JSON response.
